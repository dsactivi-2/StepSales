# Stepsales Session Handover – 2026-04-23

## 🏗 Architektur-Übersicht

Stepsales ist ein deutscher AI-Telesales-Agent für Multiposting (StepStone + Indeed) mit vollständiger Voice-Pipeline.

### Primary Stack (aktiv)

```
Deepgram STT (nova-3, de, PCM 16kHz, VAD: utterance_end_ms=1000)
    ↓ Text
LangGraph Orchestrator (8-Node State Machine, MemorySaver)
    ↓ Text
OpenAI LLM (chat.completions, gpt-4.1-2025-04-14)
    ↓ Text
ElevenLabs TTS (Streaming, eleven_multilingual_v2, German Voice)
    ↓ Audio
Telnyx play_audio (REST, kein Media WebSocket)
```

### Fallback
- OpenAI Realtime API (nicht implementiert, nur als Konzept)

## 🧠 LangGraph State Machine

### Nodes (8)
```
__start__ → greet → discovery → qualify → offer → objection → close → summary → __end__
                           ↑          ↓              ↓              ↓
                           └──────────┘              └──────────────┘
                              retry                    escalate
                                                            ↓
                                                       followup → summary
```

### Routing (`route_next_state()`)
- Nutzt Keyword-Matching (`if any(w in text for w in [...])`)
- **NICHT** LLM-basierte Intent-Klassifikation

### Checkpointer
- `MemorySaver()` (in-memory)
- Neustart = State verloren
- PostgreSQL-Checkpointer fehlt
